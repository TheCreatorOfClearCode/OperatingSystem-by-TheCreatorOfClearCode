# Журнал изменений

## Версия 1.3.1

### Системное
- Обновление меню GRUB
 1. Добавлена возможность входа в меню GRUB нажатием клавиши ESC

### Обновление драйвера
- Обновлён драйвер клавиатуры
 1. Добавлена поддержка клавиши Delete
- Обновлён драйвер VGA
 1. Добавлена функция vga_putc_at для вывода без движения курсора
 
### Обновление оболочки
- Обновлена функция сохранения истории команд
 1. Тепепь пустые строки после нажатия ENTER не сохраняются в истории команд
 2. Теперь при нажатии стрелки вверх из новой строки текущий ввод сохраняется
 3. Теперь при нажатии стрелки вниз из последней команды истории происходит переход к незавершенному вводу (если есть) или к пустой строке
 4. Добавлена проверка дубликатов в истории (одни и те же команды не будут сохраняться в истории несколько раз)
- Новый функционал
 1. Добавлена поддержка навигации по строке с помощью стерлок <-/->
- Исправлен баг
 1. При вводе символов в строке и превышения лимита при продолжении ввода символов курсор продолжал идти, но по пустой строке. Теперь при превышении количества символов в строке система не даёт ничего вводить
 
### Обновление плагина
-Обновлён плагин help
 1. Теперь при повторном вводе команды help загаловок с рамкой и отделяющая черта не пропадают

### Обновление интерпретатора команд
- Теперь он переименован в command_interpreter


### Сборка проекта

 1. Добавлена команда test (make test) для тестирования ОС в QEMU
