# Журнал изменений

## Версия 1.3

### Системное
- Обновление меню GRUB
 1. Новое оформление
 2. Добавлена вкладка "**Power options**", в которой можно выключить или перезагрузить устройство

### Обновлено ядро
- Обнавлены логи при запуске ядра:
 1. Вывод информации о ядре
 3. Повторная проверка **режима CPU**
 4. Проверка доступности **SMBIOS**
 5. Проверка доступности **CPUID**
 6. Проверка доступности вывода **количества ОЗУ**
- Переименование главного файла ядра
 1. Главный файл ядра **kernel.c** переименован в **main.c** и перемещён в **system/kernel/main/**

### Обновление системы запуска драйверов
 1. Добавлена возможность **остановки драйверов**
 2. Добавлены **приоритеты запуска**
 3. Обединена в единый файл **init_system.c**

### Обновление системы запуска плагинов команд
 1. Объединена в единный файл **init_system.c**

### Обновлены драйвера
- Обновлён драйвера **VGA**:
 1. Оптимизирована **прокрутка экрана**
 2. Добавлен **буффер**
 3. Добавлены возможность **выравнивания текста по центру**
 4. Добавлена возможность **корректной остановки работы драйвера** (при выключении или перезагрузки системы)
- Обновлён драйвера **клавиатуры**:
 1. Добавлена **поддержка стрелок**
 2. Добавлена поддержка **клавиши ESC**
 3. Добавлена возможность **корректной остановки работы драйвера** (при выключении или перезагрузки системы)
- Обновлён драйвер **Real Time Clock**
 1. Добавлена возможность **корректной остановки работы драйвера** (при выключении или перезагрузки системы)
- Обновлён Драйвер **I/O**
 1. Теперь он разделён на файлы **io.h** и **io.c**
 2. **Добавлен конфиг**, в который можно вписывать **порты ввода/вывода**, чтобы при **инициализации системы драйвер проверил работоспособность порта**
 3. При **инициализации системы выводит логи о доступности и работоспособность портов**

### Обновление плагинов
- Обновлён плагин "**about**"
 1. Добавлен вывод информации о **BIOS** (производитель, дата, версия)
 2. Добавлены вкладки
 3. **Информация об операционной системе** теперь находится в отдельном файле **system/about_system/os_info.h**
 4. **Другая информация о системе** теперь тоже находятся в разных файлах, которые находятся в **system/about_system/**
- Команды "**poweroff**" и "**reboot**" объединены в плагин **power_options**, который предоставляет **текстовый интерфейс**
 1. Добавлена команда "**power**" для открытия текстового интерфейса
- Обновлена команда "**help**"
 1. Добавлен текстовый интерфейс 

### Новый плагин
- installed
  - Доступные аргументы:
   1. plugins
   2. drivers
  - Пример использования:
   1. installed-выводит список доступных аргументов к этой команде
   2. installed plugins-открывает текстовый интерфейс со списком установленных плагинов
   3. installed drivers-открывает текстовый интерфейс со списком установленных драйверов  

### Обновление оболочки
- Новый функционал
 1. Теперь оболочка является отдельной программой и не запускается напрямую через **kernel.c** и является отдельным плагином
 2. Добавлена история команд
- Изменение интерфейса
 1. Добавлена подсказка об использовании команды "**help**" для вывода доступных команд
 2. Добавлена подсказка об использовании **стрелок вверх/вниз** для навигации **по истории команд**
 3. **Приветствие** и **подсказка** теперь находятся **вверху посредине экрана**

## Обновление интерпретатора команд (commands_execute)
- Новый функционал
 1. Добавлена поддержка команд с пробелами
 2. Добавлена поддержка команд с аргуметами
 
## Сборка проекта
- Улучшение вывода
 1. Улучшен вывод при компиляции проекта
 2. Улучшен вывод при очистке сборки проекта
- Добавлены новые команды
 1. make-полная сборка проекта
 2. make clean-очистка сборки проекта
 3. make kernel-собирает только ядро
 4. make iso-собирает ISO из существующего ядра
 5. make sources-показывает все исходные файлы
 6. make V=1-сборка проекта с подробным выводом логов
 7. make help-выводит доступные команды, цели сборки и информация о компиляторе
