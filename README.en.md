# 🌌 OperatingSystem by TheCreatorOfClearCode

**OperatingSystem by TheCreatorOfClearCode** is a modular, minimalistic operating system written in C# using **[Cosmos](https://github.com/CosmosOS)**.

---

## 📌 Main Features

- ⚙️ **Modular architecture** — flexible extension and functionality management
- 🖥️ **Console shell** with support for user commands and history
- 🧮 **Built-in utilities** — calculator, help, and other useful tools
- 📁 **File system**
- 💾 **Support for auto-loading and auto-stopping modules**
- 🧩 **Easy extensibility** through custom modules and utilities

## 🔧 Building the Project

### Requirements

- **Visual Studio 2022** (or later) — [download here](https://visualstudio.microsoft.com)
- **Cosmos User Kit** — [download here](https://github.com/CosmosOS/Cosmos/releases)

### Build Instructions

```bash
1. Open the solution file `OS.sln` in Visual Studio 2022
2. Build the project (shortcut: Ctrl + Shift + B)
3. Start debugging (F5 or click "Start") to launch the OS in a virtual machine
